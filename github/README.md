# GitHub Plugin

GitHub MCP server with all toolsets and insiders mode enabled for full repository management, code security, discussions, notifications, Copilot, and more.

## Configuration

This plugin requires the `GITHUB_PERSONAL_ACCESS_TOKEN` environment variable to be set with a valid GitHub Personal Access Token.

## Documentation

For more information about the GitHub MCP server, visit:
<https://github.com/github/github-mcp-server>
